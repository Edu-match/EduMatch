■ エデュマッチとは？
─────────────────────────────────────────────

エデュマッチは、学校・学習塾・教育機関と、
EdTech（教育×テクノロジー）サービスをつなぐマッチングプラットフォームです。

急速に広がる EdTech の世界では、何百ものサービスが存在しますが、
「どれを選べばいいかわからない」という現場の声が絶えません。

エデュマッチはその課題に正面から向き合い、
現場の先生・塾長が "自分たちにぴったりのツール" を見つけられる場を提供します。


────────────────────────────────────────────────────────────
  こんな方にオススメです
────────────────────────────────────────────────────────────

  ✓ ICT 活用・DX 推進を検討している学校・塾関係者
  ✓ 数あるツールを比較・検討したい教育担当者
  ✓ EdTech サービスの導入事例・口コミを知りたい方
  ✓ 自社の EdTech サービスを教育現場にアピールしたいプロバイダー


────────────────────────────────────────────────────────────
  主な機能一覧
────────────────────────────────────────────────────────────

  【探す・調べる】
  ・サービス検索・一覧          EdTech サービスをカテゴリ・キーワードで検索
  ・ランキング表示              人気・注目のサービスをリアルタイムでランキング化

  【読む・学ぶ】
  ・記事・ノウハウ              EdTech 活用方法・導入のコツを専門家が解説
  ・導入事例（ケーススタディ）  実際の学校・塾の成功事例を詳しく紹介
  ・イベント情報                セミナー・ウェビナー・展示会などの最新情報

  【つながる・検討する】
  ・いいね（お気に入り）        気になるサービスをワンクリックで保存
  ・一括資料請求              気に入ったサービスをまとめて資料請求
  ・レビュー投稿・閲覧          実際の利用者によるリアルな口コミ
  ・マイページ  閲覧履歴・お気に入り・資料請求状況を一元管理

  【AI サポート】
  ・AI チャットボット           OpenAI 搭載。ツール選びや活用方法を 24h 相談対応。教育に関する議論も可能。


────────────────────────────────────────────────────────────
  対応サービスカテゴリ（19 カテゴリ）
────────────────────────────────────────────────────────────

  AI 活用 / 保護者連絡 / 生徒管理 / 生徒集客 / 英会話
  映像授業 / 問題演習 / 学習管理システム（LMS）/ 質問対応
  プログラミング / 探求・キャリア教育 / 総合型選抜対策
  オンライン授業支援 / 家庭学習支援 / 知育・能力開発・幼児教育
  講師採用・育成・研修 / デバイス・ICT 環境構築
  コンサル・フランチャイズ・M&A / 助成金・補助金支援 / その他管理・代行


────────────────────────────────────────────────────────────
  料金は基本無料
────────────────────────────────────────────────────────────

────────────────────────────────────────────────────────────
  技術・インフラ（開発者・パートナー向け情報）
────────────────────────────────────────────────────────────

  フロントエンド  : Next.js 14（App Router）+ TypeScript + Tailwind CSS
  バックエンド    : Next.js API Routes + Prisma ORM
  データベース    : PostgreSQL（Supabase）
  認証            : Supabase Auth（メール / Google OAuth）
  AI              : OpenAI API（GPT チャットボット GPT4oを使用）
  メール          : Resend
  インフラ        : Vercel

  セキュリティ・可用性を考慮した本番グレードのスタックで構築されており、
  スケールアップにも柔軟に対応できる設計になっています。


────────────────────────────────────────────────────────────
  エデュマッチが選ばれる 3 つの理由
────────────────────────────────────────────────────────────

  1. 教育に特化したワンストップ情報源
     記事・事例・口コミ・比較・資料請求まで、
     EdTech 選定に必要な情報がすべて一箇所に集まっています。

  2. AI × データによるパーソナライズ
     学校種別・規模・課題を入力するだけで、
     AI が最適なサービスを自動でレコメンド。
     膨大な選択肢の中から "あなたの現場" に合うツールをすぐに絞り込めます。

  3. 現場目線の口コミ・導入事例
     実際に使った先生・塾長による生のレビューと導入事例を多数掲載。
     カタログスペックではわからないリアルな使用感を確認できます。
